# Introduction to expressions

**Expression** - a combination of literals, variables, operators, and explicit function calls (not shown above) that produce a single output value.

``` c++
int x{ 2 };             // variable initialized to 2
int y{ 2 + 3 };         // initialize variable y with value 5
int z{ (2 * 3) + 4 };   // initialize variable z with value 10
int w{ y };             // initialize with variable 
```
